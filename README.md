# Playful Sparkle - Demo admin login for OpenCart 4

The **Playful Sparkle - Demo admin login** extension for OpenCart 4.x+ helps you run demo stores by allowing limited admin access to selected extension settings pages using a preconfigured demo account.

In the extension configuration, you select which **demo admin user account** will be used for **demo access**, usually a **dedicated demo account**.

To make sharing fast, the extension adds a **Copy demo URL** button near the **Save button** on the extension settings page. One click copies a ready-to-paste demo link, so you can send it to clients or testers without manually building URLs.

The extension can also display a customizable notice at the top of every page:

> This is an OpenCart 4 demo store. Please do not enter any personal information here.

## Important security warning

You are responsible for selecting and configuring the demo account used by this extension. Use a dedicated demo-only admin user with the minimum permissions required for the pages you want to share, and never reuse a real administrator account.

**Playful Sparkle is not responsible for any damage, data loss, unauthorized access, or other consequences caused by incorrect demo account selection, excessive permissions or improper demo store configuration.**

## Features

- **Selectable Demo Account**: Designate a specific admin user account for demo access through the extension configuration.
- **One-Click Sharing**: Adds a "Copy demo URL" button near the Save button for instant link generation.
- **Customizable Global Notice**: Displays a notification banner at the top of every admin page to identify the demo environment.
- **Compatibility**: Integrates smoothly with OpenCart 4.x+, supporting a range of online store setups.
- **Multilingual Support**: Ready for international use with languages including العربية (ar), Български (bg), 中文(简体) (zh-cn), 中文(繁體) (zh-tw), Čeština (cs-cz), English (UK) (en-gb), English (US) (en-us), Français (fr-fr), Deutsch (de-de), Ελληνικά (el-gr), Magyar (hu-hu), Italiano (it-it), 日本語 (ja), 한국어 (ko-kr), Polski (pl-pl), Português (Brasil) (pt-br), Русский (ru-ru), Slovenčina (sk-sk), Español (es-es), and Türkçe (tr-tr).

## Installation Instructions

1. Download the latest version from this repository.
2. Log in to your OpenCart admin panel.
3. Navigate to `Extensions > Installer`.
4. Click the `Upload` button and upload the `ps_demo_admin_login.ocmod.zip` file.
5. Locate the extension in the `Installed Extensions` list and click the `Install` button.
6. Navigate to `Extensions > Extensions` and select `Modules` from the `Choose the extension type` dropdown list.
7. Locate the extension in the `Modules` list and click the `Install` button.
8. Click the `Edit` button, configure the extension parameters, and click the `Save` button to save your settings.
